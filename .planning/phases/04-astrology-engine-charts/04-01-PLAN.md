   - calculateAspects(planets: PlanetPosition[]): AspectData[]
   - Use ASPECT_DEFINITIONS from constants
   - Calculate shortest angle between planet pairs
   - Check if angle is within orb of each aspect
   - Determine if aspect is applying (getting closer)

4. src/calculator.ts:
   - calculateNatalChart(input: ChartInput): ChartData
   - Input: { date: Date, time: string | null, lat: number, lon: number, birthTimeKnown: boolean }
   - If time is null, use '12:00' (noon default per RESEARCH.md)
   - Calculate Julian Day
   - For each of 10 planets (Sun through Pluto), call sweph.calc_ut()
   - Call sweph.houses() with 'P' for Placidus
   - Calculate aspects
   - Return complete ChartData

- sweph.calc_ut(jd, planet, sweph.SEFLG_MOSEPH | sweph.SEFLG_SPEED)
- result.data = [longitude, latitude, distance, lonSpeed, latSpeed, distSpeed]
- sweph.houses(jd, lat, lon, 'P') returns { data: { houses, points: { asc, mc, vertex } } }
