    - Track `cachedContent` state: `{ today?: { content: string, generatedAt: string }, yesterday?: { content: string, generatedAt: string } }`.
    - On mount (useEffect), call `/api/horoscope/generate` as POST with `{ chartId }` to check for cached content or trigger generation.
    - When switching to yesterday: POST with `{ chartId, date: yesterdayDateString }`. If response has `unavailable: true`, set a flag so UI can show "not available" message.
    - After streaming completes (isLoading true->false transition), store the completed text in cachedContent for the selected date.
    - Export: `{ completion, isLoading, error, cachedContent, selectedDate, setSelectedDate, generateHoroscope }`.

    - Accepts `text: string` and `isStreaming: boolean` props.
    - Renders text with sentinel marker parsing: detect `[planet:KEY]text[/planet]` patterns and render planet names with colored styling (reuse the same color mapping as Oracle cross-highlighting if it exists, or define a simple planet-to-color map).
    - While streaming, show a blinking cursor at the end of text.
    - Use `prose` Tailwind classes for text formatting. Text color should be light (text-white/80 or similar) on dark background.
    - Handle paragraph breaks (split on double newlines).

    - Accepts `chartId: string` prop.
    - Uses `useDailyHoroscope(chartId)` hook.
    - Layout:

      b. Date navigation: Two buttons/tabs "Днес" (Today) and "Вчера" (Yesterday). Active tab highlighted. Yesterday tab disabled or shows "Не е наличен" if unavailable.
      c. Body: Shows `HoroscopeStream` with current content. If loading, show a skeleton/loading state with cosmic theme. If error, show Bulgarian error message.
      d. Empty state (no chart): "Първо добавете данни за раждане" with link to /birth-data.
    - Use glassmorphism card styling matching existing dashboard cards (bg-white/5 backdrop-blur border border-white/10 rounded-2xl pattern).
    - Auto-trigger generation when component mounts and no cached content exists for today.
