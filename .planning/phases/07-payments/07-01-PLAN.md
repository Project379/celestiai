1. Create `apps/web/app/api/stripe/checkout/route.ts` — POST endpoint:
   - Auth: `const { userId } = await auth()` from `@clerk/nextjs/server`. Return 401 with `{ error: "Неоторизиран достъп" }` if no userId.
   - Parse body: `const { priceId } = await req.json()`.
   - Validate priceId against allowlist: `new Set([process.env.STRIPE_PRICE_MONTHLY!, process.env.STRIPE_PRICE_ANNUAL!])`. Return 400 `{ error: "Невалидна цена" }` if not in set.
   - Look up user's `stripe_customer_id` from Supabase users table using service client, filtered by `clerk_id = userId`.
   - Create Stripe checkout session:
     - `mode: 'subscription'`
     - `line_items: [{ price: priceId, quantity: 1 }]`
     - `customer: user?.stripe_customer_id ?? undefined`
     - `customer_creation: user?.stripe_customer_id ? undefined : 'always'`
     - `metadata: { clerkUserId: userId }`
     - `subscription_data: { metadata: { clerkUserId: userId } }`
     - `success_url: ${process.env.NEXT_PUBLIC_APP_URL}/subscription/success?session_id={CHECKOUT_SESSION_ID}`
     - `cancel_url: ${process.env.NEXT_PUBLIC_APP_URL}/pricing?cancelled=true`
     - `allow_promotion_codes: true`
   - Return `{ url: session.url }`.

2. Create `apps/web/components/upgrade/PricingToggle.tsx` — client component:
   - `'use client'` with `useState` for `isAnnual` toggle.
   - Toggle switch (styled button or label) with "Месечен" / "Годишен" labels.
   - When annual is selected, show "~17% спестявате" savings badge.
   - Expose the toggle state and current priceId selection.

   - Reads `process.env.STRIPE_PRICE_MONTHLY` and `process.env.STRIPE_PRICE_ANNUAL` from server env.
   - Renders a client wrapper `PricingContent` that receives `currentTier`, `priceMonthly`, and `priceAnnual` as props.
   - PricingContent (client component, can be in same file as separate component):
     - Check URL searchParams for `cancelled=true` — if present, show subtle Bulgarian message: "Плащането не беше завършено. Опитайте отново когато сте готови."

     - If `currentTier === 'free'`: show "Отключи Премиум" button that calls `/api/stripe/checkout` with the selected priceId, then `window.location.href = data.url` to redirect to Stripe.

   - Style: Cosmic glassmorphism consistent with existing app theme (dark backgrounds, glassmorphic cards, accent colors from Tailwind theme).
